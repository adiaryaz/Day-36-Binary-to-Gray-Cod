# Convert Binary to Gray Code
A program to convert a standard binary string into its corresponding Gray code representation using the bitwise XOR operator.

## 📝 Description

This program takes a user-inputted binary number and translates it into **Gray code**. Gray code (or reflected binary code) is a specialized binary numeral system where two successive values differ in only one bit. It is widely used in digital logic and error correction to prevent spurious output during transitions.

The conversion algorithm implemented here is straightforward and uses string manipulation combined with mathematical logic:

1. The Most Significant Bit (MSB - the first digit on the left) of the Gray code is always identical to the MSB of the binary number.
2. Every subsequent bit is generated by performing an **Exclusive OR (XOR)** operation between the current binary bit and the previous binary bit (`binary_str[i - 1] ^ binary_str[i]`).

Additionally, the script includes strict input validation, using a Python generator `all(bit in '01' for ...)` to ensure the user only inputs valid binary digits before attempting the conversion.

---

## 🎯 Problem Statement

### Input:

* **Input 1:** A string representing a binary number (e.g., "1010").

### Output:

* If valid: Two formatted strings showing the original binary number and its calculated Gray code.
* If invalid: An error message stating "Invalid input. Please enter a valid binary number consisting only of 0's and 1's."

### Rules:

1. The program must accept string input from the user.
2. The program must validate the input to ensure it contains exclusively '0' and '1' characters.
3. If invalid, the program must reject the input gracefully with an error.
4. If valid, the program must pass the string to a function (`binary_to_gray`).
5. Inside the function, initialize the Gray code string with the first character of the binary input.
6. Use a `for` loop to iterate from index 1 to the end of the string.
7. During each iteration, cast the current and previous string characters to integers, apply the bitwise XOR operator (`^`), convert the result back to a string, and append it to the Gray code sequence.

## 💡 Examples

### Example 1 (Standard Conversion)

**Input:**

```
1010


```

**Output:**

```
Binary number: 1010
Corresponding Gray Code: 1111


```

**Explanation:** * 1st bit (MSB): Copied directly -> `1`

* 2nd bit: 1 XOR 0 = `1`
* 3rd bit: 0 XOR 1 = `1`
* 4th bit: 1 XOR 0 = `1`
* The final string is `1111`.

### Example 2 (Consecutive Bits)

**Input:**

```
1100


```

**Output:**

```
Binary number: 1100
Corresponding Gray Code: 1010


```

**Explanation:**

* MSB -> `1`
* 1 XOR 1 = `0`
* 1 XOR 0 = `1`
* 0 XOR 0 = `0`
* The final string is `1010`.

### Example 3 (Single Bit)

**Input:**

```
1


```

**Output:**

```
Binary number: 1
Corresponding Gray Code: 1


```

**Explanation:** The MSB is copied (`1`). The loop `range(1, 1)` is empty, so the function skips the XOR operations and returns `1`.

### Example 4 (Invalid Input)

**Input:**

```
10201


```

**Output:**

```
Invalid input. Please enter a valid binary number consisting only of 0's and 1's.


```

**Explanation:** The input validation detects the presence of the character `2`, which is not a valid binary digit. The conversion process is aborted safely.
